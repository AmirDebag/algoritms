1. Алгоритм — это строгое и понятное описание последовательности действий, выполнение которых позволяет получить решение поставленной задачи за конечное число шагов.

2. Свойства алгоритма:

Дискретность — алгоритм состоит из отдельных последовательных шагов.

Определённость — каждое действие алгоритма должно быть чётко и однозначно сформулировано.

Конечность — выполнение алгоритма обязательно завершается после определённого количества шагов.

Результативность — в результате выполнения алгоритма получается конкретный ответ или решение.

Массовость — алгоритм подходит для решения не одной задачи, а целого класса однотипных задач.

3. Отличие линейного алгоритма от алгоритма с ветвлением:

Линейный алгоритм — действия выполняются последовательно, одно за другим, без каких-либо условий и выбора.

Алгоритм с ветвлением — выполнение зависит от заданного условия: в зависимости от его результата выбирается один из возможных вариантов дальнейших действий.

4. Циклы применяются для повторения одного и того же набора команд несколько раз, если требуется многократное выполнение определённых действий.

5. Типы циклов в программировании:

Цикл с предусловием (while) — условие проверяется перед выполнением команд цикла.

Цикл с постусловием (do while) — сначала выполняется тело цикла, затем проверяется условие.

Цикл с параметром (for) — используется, когда заранее известно количество повторений или требуется перебор значений счётчика.
